# Задача «Сумма двух» (Two Sum)

## Цель работы

Изучить применение словарей для эффективного поиска элементов в массиве и реализовать решение задачи «Сумма двух» с использованием языка Python. Также выполнить тестирование программы с помощью библиотеки `unittest`.

---

## Постановка задачи

Дан массив целых чисел `nums` и целое число `target`.

Необходимо найти индексы двух элементов массива, сумма которых равна значению `target`.

Условия задачи:

- каждый элемент массива может использоваться только один раз;
- если решение существует, то оно единственное;
- индексы могут быть возвращены в любом порядке;
- если решение отсутствует, необходимо вернуть пустой список.

## Реализация алгоритма

Для решения задачи использовался словарь `seen`, в котором хранятся уже просмотренные элементы массива и их индексы.

Для каждого элемента вычисляется разность:

```python
diff = target - num
```

Если данное значение уже содержится в словаре, значит найдено решение задачи и можно вернуть индексы.

Код программы:

```python
def two_sum(nums, target):
    seen = {}

    for i, num in enumerate(nums):
        diff = target - num

        if diff in seen:
            return [seen[diff], i]

        seen[num] = i

    return []
```

---

## Описание работы алгоритма

Пусть дан массив:

```python
nums = [2, 7, 11, 15]
target = 9
```

Ход выполнения:

1. Первый элемент равен `2`.
   В словарь добавляется запись:

```python
{2: 0}
```

2. Следующий элемент равен `7`.

Вычисляется:

```python
diff = 9 - 7
```

Получаем:

```python
diff = 2
```

Число `2` уже содержится в словаре, следовательно найдено решение.

Возвращается результат:

```python
[0, 1]
```

## Тестирование

Для проверки корректности работы программы были написаны тесты с использованием библиотеки `unittest`.

Код тестов:

```python
import unittest

from two_sum import two_sum


class TestTwoSum(unittest.TestCase):

    def test_example_1(self):
        self.assertEqual(
            two_sum([2, 7, 11, 15], 9),
            [0, 1]
        )

    def test_example_2(self):
        self.assertEqual(
            two_sum([3, 2, 4], 6),
            [1, 2]
        )

    def test_example_3(self):
        self.assertEqual(
            two_sum([3, 3], 6),
            [0, 1]
        )


if __name__ == "__main__":
    unittest.main(
        argv=[''],
        verbosity=2,
        exit=False
    )
```

Были проверены следующие случаи:

- поиск пары чисел в начале массива;
- поиск пары чисел в середине массива;
- поиск одинаковых чисел;
- корректность возврата индексов найденных элементов.
